# 编程练习题

## 项目结构
- `src/`：存放所有编程题的C语言实现
- `problems/`：存放编程题的题目描述
- `output/`：存放编译生成的可执行文件

## 开发环境
- 操作系统：Windows x64
- 编译器：GCC 15.2.0
- 开发工具：Visual Studio Code

## 编译方法
使用GCC编译器编译单个文件：
```bash
gcc src/1.brother_string.c -o output/1.brother_string.exe
```

## 文件头配置
每个C文件都包含以下文件头信息：
- 项目名称
- 文件名称
- 作者
- 创建日期
- 最后修改日期
- 版本号
- 适用平台
- 编译依赖
- 功能描述
- 算法描述
- 适用场景
- 版权声明
- 变更记录

## 代码规范
- 函数命名：使用下划线分隔的小写字母
- 变量命名：使用下划线分隔的小写字母
- 缩进：使用4个空格
- 注释：使用//进行单行注释，/* */进行多行注释
- 代码风格：保持一致的代码风格，提高可读性
